The idea behind this app is to provide a one stop shop for 3 instruments:  Guitars, Ukuleles, and Piano.

You will be able to

a) Find chord formations
b) Figure out other voicings of this chord across a fretboard/octaves
c) View Scales of various types
d) Listen to what the chord sounds like
e) Arpeggiate what the chord sounds like
f) view common and uncommon progressions using the root chord selected
g) Detect a chord based on notes selected


Link to the app: https://lolpr0360.github.io/MUSIC_MATE/
